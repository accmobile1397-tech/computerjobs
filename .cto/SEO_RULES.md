# SEO Rules

## Every Public Page

- Title, description, canonical, OpenGraph  

## Structured Data (required)

- JobPosting, Organization, Breadcrumb (JSON-LD)  

## URLs

Readable paths — avoid raw IDs where possible:

```text
/companies/google
/jobs/tehran
/jobs/frontend
```

## Performance

Core Web Vitals mandatory (Phase 12 hardening).

## Technical

- SSR/SSG as appropriate  
- `sitemap.xml`, `robots.txt`  
- Image optimization  
- Programmatic SEO for jobs/taxonomy/location  

## Persian / RTL

SEO content in Persian; `lang="fa"`, RTL layout.
