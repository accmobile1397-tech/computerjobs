# RFC: Resume Builder

**Status:** Draft  
**Phase:** 5  
**Author:** Product / CTO  

## Summary

Platform-native resume builder — **no external file uploads**.

## Requirements

- One free resume per job seeker  
- Structured sections (experience, skills, education)  
- Skills linked to taxonomy module  
- PDF/export optional (Phase 5+)  
- Must work **without AI**; AI may suggest improvements  

## Non-Goals

- Upload PDF/DOCX  
- Import from LinkedIn (future RFC?)  

## Dependencies

- Phase 1 IAM (job seeker role)  
- Phase 3 Taxonomy (skills/technologies)  

## Open Questions

- [ ] Section templates per category?  
- [ ] Public resume URL for employers?
